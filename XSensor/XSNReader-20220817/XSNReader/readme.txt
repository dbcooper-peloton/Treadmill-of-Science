XSNReader.h -- interface of the 'XSNReader' XSN file parser library
version 9.0

Copyright (C) 2013-2022  XSENSOR Technology Corporation. All rights reserved.

This DLL allows parsing of XSENSOR XSN files to pull out frames on data in an IEEE 754-1985 32-bit floating point format.

See XSNReader.h for API.

-----------------------------------------
See TestApp.cpp for examine API usage in C.

-----------------------------------------
XSNReader.cs is a C# wrapper for the DLL
csharpTest.cs demonstrates use of the DLL in C#.
  Note be sure to use the correct 32/64 bit version of the DLL based on your Windows type.

-----------------------------------------
XSNReader.py is a Python3 wrapper for the DLL.
pythonTest demonstrates use of the DLL in Python3
  Note be sure to use the correct 32/64 bit version of the DLL based on your Windows type.
